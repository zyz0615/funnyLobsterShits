# OpenClaw × AI 玩具仓库
# OpenClaw × AI Toy Box

这里堆着一些在折腾 OpenClaw 和瞎玩 AI 时意外长出来的内容副产物。
This is where all the accidental side content ended up while I was tinkering with OpenClaw and messing around with AI.

本项目没有什么宏大规划，更像是一个“顺手做着做着就攒起来了”的小仓库。
This project has no grand plan — it is more like a little storage room for things that somehow piled up on their own.

## 这是什么
## What is this

这是一个收纳各种边角料、灵感碎片和奇怪产物的地方。
A place to store scraps, idea fragments, and other mildly strange creations.

它们原本不是主线任务，只是在实验过程中自己冒出来了。
They were never the main quest — they just spawned during experiments.

有些东西像草稿，有些像玩具，有些居然还能发出去。
Some of them look like drafts, some like toys, and some somehow turned out postable.

## 为什么会有这个仓库
## Why does this repo exist

因为每次想认真研究 OpenClaw 或 AI，最后总会顺手做出点别的东西。
Because every time I try to seriously explore OpenClaw or AI, I somehow end up making extra stuff on the side.

这些东西单独看很零散，堆在一起却有一种诡异的合理性。
Individually, these pieces are pretty random, but together they form a strangely reasonable mess.

与其让它们散落在各个角落，不如统一扔进这个玩具箱。
So instead of letting them scatter everywhere, I threw them all into one toy box.
